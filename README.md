# Cybersecurity Awareness Chatbot

## Project Overview
The Cybersecurity Awareness Chatbot is a GUI-based application developed in C# using Windows Forms/WPF. The system is designed to educate users about cybersecurity concepts through an interactive chatbot interface, a quiz system, and keyword-based natural language processing (NLP) simulation.

The chatbot assists users in learning about cybersecurity best practices such as phishing awareness, password security, safe browsing, and general online safety.

---

## Technologies Used
- C# (.NET Framework)
- Windows Forms / WPF
- MySQL (for database functionality if implemented)
- Visual Studio 2022

---

## Features

### Chatbot Interface
The system provides an interactive chatbot that responds to user input using keyword detection and simple NLP simulation through string matching techniques.

### Cybersecurity Quiz System
The quiz system includes multiple-choice and true/false questions. It provides immediate feedback and calculates a final score to reinforce cybersecurity knowledge.

### NLP Simulation
The chatbot identifies user intent through keyword detection such as “task”, “quiz”, “phishing”, and “password”, and responds accordingly.

### Task Assistant (if implemented)
The task assistant allows users to add and manage cybersecurity-related tasks, with optional reminders. Tasks may be stored using a MySQL database or internal data structures depending on implementation.

### Activity Log
The system records user actions such as quiz attempts, task creation, and system interactions for tracking purposes.

## How to Run the Project
1. Open the solution in Visual Studio 2022.
